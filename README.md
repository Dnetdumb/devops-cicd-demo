**Jenkins CI/CD with GitHub Integration**


Scenery:

Local Ubuntu VM
│
├── Docker
│   └── Jenkins (containerized)
│
├── GitHub Repo
│
├── Jenkins Pipeline (Jenkinsfile)
│
├── Dockerized App


1. Create a local VM (Ubuntu Server 22.04)
   |__Install Docker
      |__Pull Jenkins image and run container
         |__Setup Jenkins 
   
2. On Github Repo
   |__Create repo "devops-cicd-demo" 
      |__Push source app onto repo


3. On Jenkins Server
   |__Create project "devops-cicd-demo"
